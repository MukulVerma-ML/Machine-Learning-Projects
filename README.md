# Machine Learning Projects

This repository contains all my Machine Learning projects.

## Project 1: Student Performance Prediction
**Problem Statement**: Predict a student's Performance Index based on Hours Studied, Previous Scores, and Sleep Hours.

**Tech Stack**: Python, Pandas, Numpy, Scikit-learn

**Models Used**: Linear Regression, Ridge, Lasso, Logistic Regression

**Dataset**: `Student-Performance/Student_Performance.xlsx`

**How to Run**: Click the "Open in Colab" button in `Student-Performance/Student.ipynb`
